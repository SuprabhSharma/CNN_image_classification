📌 Project Summary

This project implements an Image Classification model using a Convolutional Neural Network (CNN) built with TensorFlow and Keras. The objective of the project is to train a deep learning model that can automatically learn visual features from images and classify them into their respective categories.

The workflow begins with data loading and preprocessing, where image data is normalized and prepared for training. A CNN architecture is then constructed using multiple layers including Conv2D, MaxPooling, Flatten, and Dense layers to extract spatial features and perform classification.

During training, the model learns hierarchical patterns from images, enabling it to identify complex visual structures. After training, the model is evaluated using performance metrics such as accuracy, confusion matrix, and classification report to measure its prediction capability.

This project demonstrates the practical implementation of deep learning for computer vision, covering the complete pipeline from data preparation to model evaluation.
